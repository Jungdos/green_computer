1. 배열 - 여러 개의 변수를 한꺼번에 담을 수 있는 유형 [ ] 
ex ) var 변수명 = [값1,값2,값3]
     index 호출 시 : 0부터 호출
     length 속성 : 배열요소의 갯수가 몆개인지 알려준 것(값1,값2,값3 총 3개)

2.for (랙) (초기식; 조건식; 종결식;) 문 {실행문;}
	
초기식 : 임의의 변수 지정(i로 많이들 지정하는 이유? index의 갯수 위주로 작업하기 떄문에!)

조건식 : 몆번을 실행할지를 정해주는것.
ex) var i = 0; i=5  <= 가 들어가는 이유 0부터 시작이기 떄문에

종결식 : 반복할 표현식을 실행후 변수값이 바뀔 식
ex) i++ , i-- 증감식,감소식 

3. for in문 - 배열이나 객체처럼 나열된 값을 쉽게 반복할 수 있는 구문
ex)for(var i in 배열명) {반복할 표현식} 

4.break키워드 : switch 조건문이나 반복문을 벗어날 떄 사용
(작업을 할떄 브레이크가 들어가면 거기서 종료 시키는것.)

5. continue키워드 : 반복문 내에서 현재 반복을 멈추고 다음 반복을 진행
(내가 가지고있는걸 반복했을때 컨티뉴가 걸려있다면 현재 반복을 멈추고 다음것으로 넘어가는것)

6.setTimeout(funtion(){ },초) : 초를 계산하고 실행을 진행한다.
1초 = 1000

7.setInterval(funtion( ){ },초) : 초 단위별로 실행한다.
(다른사람껄 수정보완할수있을정도의 실력)
